# Sistema de Gestión para Nano Florería

## Descripción
Este proyecto consiste en el diseño de una base de datos para una florería, con el objetivo de administrar clientes, pedidos, arreglos florales e inventario. El sistema permitirá organizar la información y facilitar el control de ventas.

## Motivación
Muchas florerías pequeñas llevan registros manuales que pueden generar errores o pérdida de información. Este sistema busca mejorar la organización, el control del inventario y el seguimiento de pedidos.

## Entidades principales
- Cliente
- Flor
- Arreglo
- Pedido
- Detalle_Pedido

## Diagrama Entidad-Relación
El modelo contempla la relación entre clientes y pedidos, así como el control de productos incluidos en cada compra.
